# Magit

前提：Spacemacs vim mode

## magit-status

    SPC g s

Display information about the current Git repository.

**Most Magit commands are commonly invoked from the status buffer.**

## In status buffer

### magit-refresh

    g r

刷新缓冲区

### magit-stash-file

    # move to unstaged file
    s

Stage untracked file.

### magit-unstage-file

    # move to staged file
    u

Unstage staged file.
